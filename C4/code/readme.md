5. 复杂度分析
    1. 邻接矩阵的建立时间复杂度是O(e+n)，因为需要输入n个顶点和e条边，需要的存储空间为O(e+n^2)。邻接表的建立时间复杂度是O(n+e)，需要输入n个顶点和2e条边。需要的存储空间为O(e+n^2)
    3. 深搜和广搜的时间复杂度都是O(n+e)，只需遍历每个顶点和最多两次每条边
    4. 时间复杂度仍为O(n+e) ,只需遍历每个顶点一次和每条边两次
